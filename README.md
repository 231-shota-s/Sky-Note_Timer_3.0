# Sky Note Timer 3.0

ミニマリズムと堅牢な機能性を追求した、ブラウザベースのプロフェッショナル・タイマー。
余計な依存関係を排除し、バニラJSのみで構築された超軽量・高速動作のツールです。

## 💎 プロフェッショナル機能

### 1. ユーザーインターフェース
- **直感的テンキー入力**: 物理キーボードおよび画面上のUI両方から、思考を妨げない素早い入力が可能。
- **フルスクリーン・アダプティブ**: `F`キー一つで、あらゆるアスペクト比のディスプレイにおいて視認性を最大化する「集中モード」へ移行。
- **ダイナミック・進捗リング**: 残り時間を視覚的に把握できるSVGアニメーション。

### 2. テクニカル・パフォーマンス
- **高精度タイマー・エンジン**: `setInterval` の誤差を補正するアルゴリズムを採用し、バックグラウンドでの安定した計測を実現。
- **Screen Wake Lock API**: 長時間の計測や全画面表示中でも、デバイスのスリープを自動的に防止（対応ブラウザのみ）。
- **ゼロ・トラッキング**: 外部サーバーへの通信、広告、解析タグを一切含まず、完全なプライバシーとオフライン動作を保証。

### 3. カスタマイズ
- **サウンド・セレクション**: 使用環境に合わせた3種類の高音質アラート（Web Audio APIによる合成音）。
- **背景点滅アラート**: 音を出せない環境でも視覚的に通知を認識可能。
- **バイリンガル対応**: 日本語と英語の完全切り替え。

## ⌨️ キーボード・ショートカット
| Key | Action |
| :--- | :--- |
| `Space` | タイマーの開始 / 一時停止 |
| `0 - 9` | 数値の入力（待機時） |
| `C` / `Backspace` | 入力内容のクリア |
| `R` | 完全リセット（全設定をデフォルトへ） |
| `F` | 全画面表示のトグル切り替え |
| `Esc` | ヘルプモーダルの終了 |

## 🛠 技術スタック
- **Language**: HTML5, CSS3, JavaScript (ES6+)
- **Graphics**: SVG (Scalable Vector Graphics)
- **Audio**: Web Audio API (振幅・周波数制御)
- **System**: Screen Wake Lock API
